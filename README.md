# Movie App UIKit

Aplikasi iOS sederhana untuk menampilkan daftar movie.

---

## Cara Menjalankan
1. Clone repository ini  
2. Buka project di Xcode (`.xcodeproj`)  
3. Jalankan menggunakan simulator atau iPhone device  

---

## Pendekatan
Aplikasi ini dibuat menggunakan UIKit dengan pola MVC sederhana dan mengambil data dari API menggunakan URLSession.

---

## Fitur Utama
- Menampilkan list movie  
- Menampilkan detail movie  
- Mengambil data dari API
